# Ascendency System v1 (5e-Compatible)

## Core Mechanic
- Divine Core awakens in session one.
- Progression tiers:
  - Awakened (1-4)
  - Rising (5-10)
  - Exalted (11-16)
  - Apex (17-20)
- Ascendency is an alternate feat track and does not consume normal ASI/feat picks.

## Balance Guardrails
- Damage riders from Ascendency features should usually trigger once per turn.
- Hard control effects must include either a save, duration cap, or immunity window.
- Defensive and healing features should preserve class identity rather than replace it.
- Pathfinder-style stacking bonuses are intentionally avoided; use bounded bonuses.

## Path Themes and Roles
- **Frontline pressure:** War, Bone, Ruin, Chain
- **Skirmish and burst:** Storm, Wild, Dusk
- **Control and tempo:** Veil, Mirror, Tide
- **Support and command:** Forge, Dawn, Star
- **High-risk predation:** Hunger

## Progression Pattern
- **Awakened (1-4):** one identity-defining minor feature.
- **Rising (5-10):** one major tactical feature plus one enhancement node.
- **Exalted (11-16):** encounter-shaping signature feature.
- **Apex (17-20):** mythic capstone with strict usage limits.
