# Session One: The Breaking of Kharad-Vul

## Goal
Run an escape-focused opening where the party survives the first night of siege, witnesses an Underscar breach, and awakens Divine Cores during a high-stakes choice.

## Suggested Scene Flow
1. Alarm Bells in the Lower Forges
2. Breach at Chainwell Gate
3. Crownflame Ignition (awakening)
4. Escape Through the Ash Lift

## Session Outcome
- Milestone to level 2
- Each player gains an awakened path affinity
- Each player receives one faction attention tag
